# Far From Home

Far From Home is a mobile based house rental application developed using Flutter Framework which is mainly for students, employees and other professionals who are living as PGs or on rent and also for the property owners who are searching for a platform to post their property on rent. The sole purpose of this application is to achieve complete transparency between the owner and the tenant in the entire process of renting by removing the role of a middleman â€“ a broker. Using this application user can search the property, rent the property, contact the owner/tenant, pay rent using integrated payment API and generate the invoice for your transaction,etc.  

[Download Project Video](https://github.com/KevinPatel04/Far-From-Home/blob/master/Far%20From%20Home.mp4)
